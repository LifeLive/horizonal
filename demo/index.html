What is it?

Horizonal.js takes any static HTML document and transform it into a collection of discrete "pages". The transitions between one page an the next can then be programmed via CSS transforms, CSS animations, JavaScript, or any combination of these.

Disable Horizonal with the switch above and you'll see that this is just a regular old HTML document. Try out some of the themes to get a taste of the kind of effects that can be achieved.

What's It For?

Horizonal.js grew out of some simple experimentation with CSS transforms and animations, and if nothing else it is a showcase for the cool visual effects that can be achieved through these technologies. Other use cases could be slides for presentations, image galleries, portfolios, marketing websites, or whatever else you can think of.

Features

  • Fully customizable - limited only by your imagination (and JavaScript & CSS chops)
  • Touch-enabled: page through the document by swiping.
  • Can handle URL fragments - e.g. go to that poem
  • Can handle large elements that span multiple pages - e.g. this huge photo
