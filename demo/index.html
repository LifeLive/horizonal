What is it?

Horizonal.js takes any static HTML document and transform it into a series of discrete "pages". The transitions between one page an the next can then be programmed via CSS transforms, CSS animations, JavaScript, or any combination of these.

How?

You specify which elements you wish to display via the familiar jQuery selector syntax. Horizonal will then traverse the document, splitting it up into a series of pages, and figuring out which element should go where.

Features

  • Fully customizable - limited only by your imagination (and JavaScript & CSS chops)
  • Can handle URL fragments - e.g. go to that poem
  • Can handle large elements that span multiple pages - e.g. this huge photo
